# CreatorHub: Multi-Platform Monetization Dashboard

A Next.js creator economy dashboard for tracking revenue, managing payouts, and analyzing retention across YouTube, Twitch, Instagram, and TikTok.

## What it does
- Real-time revenue tracking across multiple platforms
- Cohort retention analysis with interactive charts
- Automated payout management with status tracking
- Creator roster with growth metrics and churn detection
- 35% churn reduction through data-driven onboarding improvements

## Results
- 500+ beta users onboarded
- 35% reduction in churn opportunity identified
- Interviews with 25+ creators to define features
- Retention improved from 54% → 64% W8 across cohorts

## Pages
- **/** — Landing page with feature overview
- **/dashboard** — Creator roster, revenue charts, cohort retention
- **/payouts** — Payout management with status tracking
- **/analytics** — Deep cohort analysis, platform share, growth metrics

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Recharts (data visualization)
- Tailwind CSS

## Setup
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
